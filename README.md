# 串口配置工具使用说明

## 1. 程序概述

本程序是一个基于Qt开发的串口配置工具，用于管理和配置串口设备。主要功能包括设备搜索、连接管理、网络配置、串口参数设置、工作模式配置等。

## 2. 系统要求

- 操作系统：Linux
- Qt版本：5.12.10或更高
- 依赖库：Qt Widgets、Qt Network、Qt GUI

## 3. 安装方法

### 3.1 从源码编译

1. 确保系统已安装Qt开发环境
2. 进入项目目录：
   ```bash
   cd /home/sxb/zongPro/chuankou20260319_linux_new
   ```
3. 执行qmake生成Makefile：
   ```bash
   qmake
   ```
4. 编译项目：
   ```bash
   make
   ```
5. 运行程序：
   ```bash
   ./chuankou
   ```

### 3.2 直接运行

如果已经有编译好的可执行文件，可以直接运行：
```bash
./chuankou
```

## 4. 使用方法

### 4.1 设备搜索

1. 打开程序后，在主界面点击「搜索设备」按钮
2. 程序会自动搜索局域网内的设备，并在表格中显示搜索结果
3. 搜索结果包括设备状态、型号、序列号、名称、固件版本、设备状态、IP地址、MAC地址等信息

### 4.2 设备连接

1. 在搜索结果表格中选择要连接的设备
2. 点击「连接」按钮
3. 连接成功后，界面会切换到设备管理页面

### 4.3 网络配置

1. 在设备管理页面点击「基本设置」按钮
2. 在网络配置界面，可以设置设备的网络参数，包括：
   - LAN1 IP地址、子网掩码、网关
   - LAN2 IP地址、子网掩码、网关
   - DHCP设置
3. 点击「应用配置」按钮保存设置

### 4.4 串口参数配置

1. 在设备管理页面点击「串口设置」按钮
2. 在串口配置界面，可以设置：
   - 端口号
   - 波特率
   - 数据位
   - 停止位
   - 校验位
   - 流控
   - 接口类型
3. 点击「应用配置」按钮保存设置

### 4.5 工作模式配置

1. 在设备管理页面点击「模式设置」按钮
2. 在工作模式配置界面，可以设置：
   - 端口工作模式（TCP Server Mode、Real COM Mode、Redundant Mode）
   - TCP活动检测时间
   - TCP最大连接数
3. 点击「应用配置」按钮保存设置

### 4.6 端口管理

1. 在设备管理页面点击「端口管理」按钮
2. 可以进行以下操作：
   - 端口自检
   - 端口重置
   - 端口镜像
   - 端口锁定/解锁
   - 查看端口连接信息
   - 查看端口状态信息
   - 查看端口错误信息

### 4.7 设备管理

1. 在设备管理页面点击「设备管理」按钮
2. 可以进行以下操作：
   - 设备锁定/解锁
   - 设备日志查看
   - 看门狗设置
   - 恢复出厂设置
   - 重启设备
   - 固件升级

## 5. 功能说明

### 5.1 设备搜索与连接
- 支持UDP广播搜索局域网内的设备
- 支持TCP连接设备进行配置管理

### 5.2 网络配置
- 支持静态IP和DHCP两种网络配置方式
- 支持同时配置LAN1和LAN2接口

### 5.3 串口配置
- 支持配置16个串口的参数
- 支持RS-232、RS-422、RS-485-2等接口类型

### 5.4 工作模式配置
- 支持TCP Server Mode、Real COM Mode、Redundant Mode三种工作模式
- 支持配置TCP连接参数

### 5.5 端口管理
- 支持端口自检和重置
- 支持端口镜像功能
- 支持端口锁定和解锁
- 支持查看端口状态和错误信息

### 5.6 设备管理
- 支持设备锁定和解锁，提高设备安全性
- 支持看门狗设置，增强设备稳定性
- 支持固件升级，方便设备维护
- 支持恢复出厂设置，解决配置问题

## 6. 故障排除

### 6.1 设备搜索失败
- 检查设备是否在同一局域网内
- 检查设备是否已开机
- 检查网络连接是否正常

### 6.2 设备连接失败
- 检查设备IP地址是否正确
- 检查设备是否已被其他程序连接
- 检查网络连接是否正常

### 6.3 配置保存失败
- 检查设备是否处于正常工作状态
- 检查网络连接是否稳定
- 检查配置参数是否合法

## 7. 注意事项

1. 设备锁定后需要解锁码才能解锁，请妥善保管解锁码
2. 启用看门狗后，若未按时喂狗，设备将循环重启
3. 固件升级过程中绝对不能断电或断开连接，否则可能导致设备损坏
4. 恢复出厂设置会清除所有配置，请谨慎操作
